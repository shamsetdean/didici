Chaînes bloquées
Les vidéos issues de ces chaînes sont automatiquement filtrées des résultats.
Le nom doit correspondre exactement au nom de la chaîne YouTube (ex : "RMC Découverte"). La comparaison est insensible à la casse.
À propos
DocuScope utilise l'API YouTube officielle. Aucune vidéo n'est hébergée. Les publicités YouTube dans le lecteur sont celles de YouTube et de ses partenaires — elles ne sont ni bloquées ni modifiées par ce site.

Quota API : 10 000 unités/jour (gratuit Google).

Mentions légales

Éditeur

Site personnel à vocation éducative, non commercial. Aucune donnée personnelle collectée ou transmise à un tiers.

Hébergement

GitHub Pages — Microsoft Corporation.

Contenus vidéo

Toutes les vidéos proviennent exclusivement de YouTube via l'API officielle YouTube Data API v3 et le lecteur embed YouTube. Ce site n'héberge aucune vidéo. Les droits appartiennent à leurs auteurs respectifs.

Publicités YouTube

Ce site ne bloque pas et ne modifie en aucune façon les publicités diffusées dans le lecteur YouTube intégré. Celles-ci relèvent exclusivement de YouTube et des ayants droit.

Conformité YouTube

Ce site respecte intégralement les conditions d'utilisation de YouTube et les conditions de l'API YouTube.

Données locales

Seul localStorage est utilisé pour mémoriser les favoris et les préférences. Aucun cookie tiers, aucun tracker, aucun serveur externe.
