Mentions légales

Éditeur

Site personnel à vocation éducative, non commercial. Aucune donnée personnelle collectée ou transmise à un tiers.

Hébergement

GitHub Pages — Microsoft Corporation.

Sources vidéo

Les vidéos proviennent de deux sources officielles : YouTube via l'API officielle YouTube Data API v3 (lecteur embed YouTube), et Internet Archive via son API publique gratuite. Les contenus Internet Archive sont dans le domaine public ou sous licence Creative Commons. Ce site n'héberge aucune vidéo.

Publicités

Ce site ne bloque pas et ne modifie pas les publicités du lecteur YouTube. Internet Archive ne diffuse pas de publicités.

Signalement de contenus illicites

Conformément à la loi française et au règlement DSA, tout utilisateur peut signaler un contenu illicite. Les signalements peuvent être transmis à l'Arcom.

Données locales

Seul le localStorage est utilisé — profil, favoris, historique. Aucun cookie tiers, aucun tracker.

Conformité YouTube

Ce site respecte les CGU YouTube et les conditions de l'API YouTube.
