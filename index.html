Mentions légales

Éditeur

Site personnel à vocation éducative, non commercial. Aucune donnée personnelle collectée ou transmise.

Hébergement

GitHub Pages — Microsoft Corporation.

Contenus vidéo

Toutes les vidéos proviennent de YouTube via l'API officielle YouTube Data API v3 et le lecteur embed YouTube. Ce site n'héberge aucune vidéo.

Publicités YouTube

Ce site ne bloque pas et ne modifie pas les publicités du lecteur YouTube.

Signalement de contenus illicites

Conformément à la loi française et au règlement DSA, tout utilisateur peut signaler un contenu illicite. Les signalements peuvent être transmis à l'Arcom.

Données locales

Seul le localStorage est utilisé — profil, favoris, historique. Aucun cookie tiers, aucun tracker.

Conformité YouTube

Ce site respecte les CGU YouTube et les conditions de l'API YouTube.
